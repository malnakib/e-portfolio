Module 1: Foundations of Software Engineering

This module focuses on the principles and practices of software engineering. It equips students with key skills in project planning, system requirements, UML modeling, and teamwork under Agile methodologies.

Learning Outcomes

  • Understand the stages of the software development lifecycle.
  • Apply software design principles and documentation practices.
  • Develop artefacts like requirement specs, wireframes, and models.
  • Reflect on team collaboration and professional growth.

Reflection

Through this module, I learned how to formalize and structure project ideas using UML, wireframes, and requirement specifications. It sharpened my collaboration and planning skills, laying a strong foundation for future technical modules.
